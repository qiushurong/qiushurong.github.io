软件-文件加密解密工具[克捷公司]

发表于 2014-02-24   |  字数 811   |  预计阅读 4分钟   |   分类于 Java SE   |  

2013年暑假接的一个小项目，赚了一丢丢钱，勉强生活费吧，但是怎么说也是我为数不多的项目之一。

某机顶盒项目的附属模块，用于对相关素材进行加密处理，主要是视频文件的加密，以及对于客户端上的解密播放，防止有心之人盗用。类似于目前主流的视频客户端下载文件的缓存加密功能。

本附属程序属于克捷公司所有，作者邱树荣。未经过授权不允许用于商业手段。
软件采用java SE编写
