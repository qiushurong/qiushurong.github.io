Java5枚举类型高级用法之抽象函数

发表于 2014-02-21   |  字数 116   |  预计阅读 1分钟   |   分类于 Java SE   |  

使用JAVA5提供的枚举类型 - 抽象函数让子类去覆盖抽象方法 - 实现特定的功能。

/**
 * 使用JAVA5提供的枚举类型 - 抽象函数
 * 让子类去覆盖抽象方法 - 实现Next()的功能
 * @author qsuron - 博客:http://blog.csdn.net/qsuron
 */
public class Java5 {
    public static void main(String[] args) {
        Week w = Week.MON;
        System.out.println(w.next());
    }
    public enum Week{
        //由于麻烦,只写2个星期数
        MON{
            public Week next(){
                return TUE;
            }
        },
        TUE{
            public Week next(){
                return MON;
            }
        };
        public abstract Week next();
    }
}
