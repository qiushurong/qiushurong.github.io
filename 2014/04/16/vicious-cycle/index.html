[无聊纯扯淡]深谈程序员的恶性循环

发表于 2014-04-16   |  字数 292   |  预计阅读 1分钟   |   分类于 心情杂谈   |  

如果有一天真的陷入这么一个循环，我们能做些什么呢？
 

穷人的恶性循环：
穷 -> 需要努力工作 -> 没有时间去交际 -> 人脉越来越狭窄 -> 工作越来越难做 -> 越需要努力去工作 -> 越没有时间去发展人脉 -> 越穷

富人的良性循环：
有钱 -> 工作很轻松 -> 很多时间都在交际上 -> 人脉越来越广 -> 工作越来越不用努力 -> 越有更多的时间精力去发展人脉 -> 越富有

然而，我们程序员也如此！

加班 -> 没空学习 -> 老是写同等水平代码 -> 无法提升代码质量 -> 老是出BUG -> 老是需要修改 -> 加班 -> ….

是上班之余压榨时间学习？

还是百度代码COPY粘上？

还是期盼问题代码不出BUG？

恐怕让公司另寻能人会比较简单。
