Eclipse-编辑器皮肤设置方法&自制主题分享

发表于 2014-07-14   |  字数 893   |  预计阅读 4分钟   |   分类于 通用开发技术   |  

编程是一个体力活啊，又很枯燥乏味，给自己的eclipse加点料吧，这样多少还能看着新鲜点，不行过两天再换一个皮肤~还能保护眼睛，一举多得啊~

1 自制主题分享

小树自己制作了一个主题 - qsuron，顺带上传上来，喜欢大家喜欢！

下载地址：百度网盘 ， 皮肤预览：(点击放大)

 

使用方法看下面 第2点 安装主题插件(没有装插件是不可以使用任何主题的)，以及第3点 主题导入。

 

2 主题插件的安装

其实很简单，现在一起来：

步骤：

-> 在菜单栏点击Help，选择Install New Software，点击 add，输入插件信息

-> name : 随便输入

-> Location : http://eclipse-color-theme.github.com/update

点击ok之后，出现这样的画面

如果提示找不到，请下载：http://eclipse-color-theme.github.io/update/eclipse-color-theme-update-site.zip，然后在上一步中，点击attach手动添加。

注意：这里有一个复选框：Contact all update sites … ，建议不要勾选！勾选了要耗费大量时间！

 

点击 Select All , next , next 。然后Eclipse会提示是否重启（选择YES）。待重启完成之后：打开菜单：Windows ->Perferences ->General-> Appearence ->Color Theme

就可以选择喜欢的皮肤了。

这么多皮肤，随便挑吧，这里我就挑了一个颜色比较花哨的。

 

3.自制皮肤的导入

 

导入皮肤：

除了这些插件自带的皮肤，网上还有许多网友制作的皮肤，我们可以下载下来之后导入到eclipse中，导入方法

步骤：

同上面的选择皮肤，到达 Color Theme 设置页面后，点击下面的 Import theme 即可导入皮肤！
